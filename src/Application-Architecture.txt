Приложение АвтоПарк

Обьекты
1) Автопарк - содержащее все машины, количество, - Паттерн синглтон потому что автопакр всегда один
2) Супер класс машина - у всех машин есть ( тип(седан), модель(бмв), название(f90), цвет(черный), Руль((слева)), обьем(4,4), топливо(бензин), бак(40литров), кпп((механика)),
 привод((передний)), лошадинная сила(720), ( комплектация(массив строк ((люк [Имеестся])) (люк) (шторы) (диски) (карбон) (выхлоп) ) ) )

Класс Клиент - может иметь только одну машину

3) Машина Бмв ( обекты (f90, g30, x5, x7) )
4) Машина Mersedes ( обьекты (w222, w223, cls) )
5) Машина Porshe ( (911, 991sturbo, panamera) )

Интерфейсы для машин
1) (Разгон машины от 0-100)
2) (Давать боком (идеально для седанов и не лучше для больших))
3) (Пройтись по не ровным дорогам (идеально для больших и не лучше для седанов или ужас для спорт машин))
4) Опустошение бака ( (Разгон - 3), (боком - 2), (Пройтись по не ровным дорогам - 2))
5) Пополнение бака в полную



Логика
Автопарк отображает все машины пользователь выберает одну и выводится все информация о машина, далее клиент либо берет либо нет! У клиента имеется только
одна машина, и затем отображаются действия (Интерфейсы)
